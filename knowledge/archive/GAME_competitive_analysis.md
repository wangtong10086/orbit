## HEX 策略 (Top: 67%)

### 先后手胜率

| 棋盘 | P0先手 | P1后手 | P0开局 |
|------|--------|--------|--------|
| 5×5 | **100%** | **0%** | 永远c3 |
| 7×7 | **93%** | 47% | 永远e3 |
| 9×9 | **94%** | 70% | d6(主), f4 |
| 11×11 | 83% | 60% | f6, d7 |

### 赢棋核心: 对角链
- chain_ratio(相邻移动比) = 0.75 — 每4步3步与前一步相邻
- P0: 中心开局 → 朝两角延伸 → 对手无法同时阻断

### P0必胜路线

**5×5**: c3 → b4 → a5 → d2 → e1 (主对角线直连)
**7×7**: e3 → d4 → c5 → b6 → b7 → f2 → g1
**9×9**: d6 → e5 → f4 → g3 → h2 → i1 → d7 → c8 → b9
**11×11**: f6 → e7/g5 双向延伸

### P1策略
- 5×5后手必输（数学证明）
- 7×7+: 对角链 chain_ratio ≥ 0.83 才能赢

## LIARS DICE 策略 (Top: 29%)

### 概率基础
- 每骰子匹配某面值概率 = 1/3 (本身1/6 + wild6 1/6)
- 对手5骰期望匹配数 = 1.67

### 叫Liar决策表

| 需对手有 | 概率 | 决策 |
|---------|------|------|
| ≤1个 | >83% | **绝不叫** |
| 2个 | ~73% | **不叫** |
| 3个 | ~35% | 谨慎叫 |
| ≥4个 | ~11% | **叫Liar** |
| ≥5个 | ~1.3% | **必须叫** |

### 开局出价
开局bid = 我最多面值数(含wild6) + 1~2

### 致命错误
1. **OVERBID**: 出价超过 我的数量+3 → 必被叫Liar
2. **错误叫Liar**: 对手只需2个时叫Liar → 73%概率输

### 赢棋模式
100%赢局都包含叫Liar。流程: 合理bid → 等MCTS raise → 判断是否合理 → 叫Liar

## OTHELLO 策略 (Top: 48%)

### 先后手
- P0(黑先手): 60%胜率
- P1(白后手): 36%胜率

### 最强开局

**P0**: f5→g3→c3 (75%), f5→c3→e3 (67%), c4→b6→f6 (75%)
**P1**: f6→g5→d3, f6→f4→c6, c3→d2→f5

### 策略要点
- LLM偏好角落(a1,a8,h1,h8) — 基础正确
- MCTS更注重中路控制(e3,e2,c5) — 更高级
- 不要太早抢角，控制中路节奏

## CLOBBER 策略 (Top: 18%)

### 开局决定胜负

| 开局 | 胜率 | 评价 |
|------|------|------|
| **c5c4** | **48%** | 最优！中心进攻 |
| e4d4 | 43% | 中等 |
| a2a3 | **11%** | 最差！角落逃跑 |
| c3c4 | **0%** | 禁止 |
| f2f3 | **0%** | 禁止 |
| e3e4 | **0%** | 禁止 |

### 赢棋规律
- 赢局平均16.8步，输局22.7步 — **短游戏=赢**
- 赢局中心区域走子51% vs 输局39%
- 赢局向下吃(进攻) 41% vs 输局25%

### 策略
1. 永远c5c4开局
2. 向中心收拢，不向边缘扩散
3. 最大化自己后续moves数，最小化对手

## SFT数据生成关键要求

### 格式
**assistant回复只有一个数字，无think链，无解释。**

```json
{"role": "assistant", "content": "42"}
```

### 数据来源
- **hex**: MCTS 50000sim自对弈 + 已知opening book
- **liars_dice**: 概率计算最优决策（不用MCTS）
- **othello**: MCTS 100000sim 或 Edax引擎
- **clobber**: MCTS 200000sim，禁止边缘开局

### 数据量建议
| 游戏 | 对局数 | (state,action)对 |
|------|--------|-----------------|
| liars_dice | 5000 | 15000 |
| hex 5×5 | 500 | 2500 |
| hex 7-11 | 6000 | 56000 |
| othello | 5000 | 150000 |
| clobber | 10000 | 170000 |
