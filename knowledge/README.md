# Knowledge Base

This directory contains non-normative research notes, environment analyses, and
working reference material.

Use this directory for:

- deeper environment-specific notes
- historical data and training analysis
- failure analysis and strategy notes

Do not use this directory as the primary source of truth for:

- current CLI behavior
- current architecture boundaries
- current operational requirements

For those, use:

- [../README.md](../README.md)
- [../docs/README.md](../docs/README.md)

## Directory Layout

- `environments/`
  - environment-specific reference notes and analyses
- top-level files
  - data, infra, scoring, training, failures, and strategic notes

## Document Status

Unless a file explicitly says otherwise, treat documents here as:

- status: reference
- audience: maintainers and contributors
- authority: non-normative

These notes may be useful, but they can drift behind the active codebase and
docs set.
