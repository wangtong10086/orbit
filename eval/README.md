# Evaluation Archive

This directory stores archived evaluation reports and experiment result
summaries.

Use this directory for:

- reading historical benchmark outcomes
- comparing past model iterations
- reviewing experiment-specific analysis

Do not use this directory as the current product or architecture specification.

## Conventions

- each subdirectory represents a report batch or model version
- report files are archival records, not normative documentation
- newer reports do not automatically invalidate older ones unless they say so
