# NW v2.16 得分分析 — 43个低分任务是最大提升机会

## 当前: 35.46（96% non-zero）

最大瓶颈: 43个任务得分 0.01-0.24（平均 0.15）。如果提到 0.50 → NW 可达 50+。

## 低分任务特征（vs 高分对比）

低分任务和高分任务的工具调用次数一样（~10次），差异在质量：

1. **任务类型偏商务/预算**: 低分任务多是"商务出差+预算限制+多交通方案对比+费用明细"。高分任务多是"家庭旅游/主题游"（灵活度高）
2. **冗余搜索**: 低分任务平均 around_search 3.1次/任务（高分仅 0.5次），说明搜索不精准需要反复尝试
3. **规划不完整**: 任务要求"对比2-3种交通+酒店+餐厅+费用明细"但模型只完成部分
4. **结构化输出差**: 缺少费用表格、方案对比格式

## 需要的训练数据

1. **商务差旅场景** 50-100 条: 含预算约束、交通方案对比（航班/高铁对比表格）、费用明细
2. **结构化输出格式**: 表格式费用对比、分项明细、方案 ABC 对比
3. **精准搜索示例**: 展示一次 poi_search 直接命中，而非多次 around_search 模糊搜索
4. **完整规划示例**: 一个任务中完整覆盖交通+住宿+餐饮+费用的全流程
