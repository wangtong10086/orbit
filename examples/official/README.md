# Official Examples

This directory contains versioned example configs for supported workflows.

Current official example:

- [`training/targon-qwen3-32b-full-sft.yaml`](training/targon-qwen3-32b-full-sft.yaml)
  - one-command training launch
  - Hugging Face dataset download
  - Hugging Face model repo creation
  - isolated Targon rental provisioning
  - control-plane submission through `forge control launch train`

Copy and edit the example before use.
